## When DDS Genuinely Can't Meet Your Deadline

If DDS cannot deliver before your deadline, the **Interface-First Emergency Protocol** applies:

<InfoBox variant="warning">
  <strong>Validation Status: Proposed Process</strong> — This protocol has been designed but not yet
  validated in production. It represents our intended approach. We will update this section with
  real examples once the first consumer goes through this flow.
</InfoBox>

<InfoBox variant="tip">
  <strong>Key Principle:</strong> DDS is the single source of truth for UI interfaces.
  Even in emergencies, DDS provides the interface spec first, then everyone builds to it.
  This eliminates the "migration twice" problem.
</InfoBox>

<div style={{
  background: PRIMITIVES.white,
  borderRadius: RADIUS.lg,
  padding: '24px',
  border: `1px solid ${SLATE[200]}`,
  marginBottom: '24px',
}}>
  <div style={{ display: 'flex', flexDirection: 'column', gap: '12px' }}>
    {[
      { step: '1', action: 'Notify', desc: 'Tell architect: "I need X by [date], DDS can\'t deliver full implementation"' },
      { step: '2', action: 'Interface', desc: 'DDS provides component interface spec within 24-48h (props, data shapes, API contract)' },
      { step: '3', action: 'Review', desc: 'Backend devs review interface, provide feedback if API concerns exist' },
      { step: '4', action: 'Build', desc: 'Consumer builds to DDS interface spec; Backend builds API to same spec' },
      { step: '5', action: 'Ship', desc: 'Consumer deploys with stub/basic implementation' },
      { step: '6', action: 'Swap', desc: 'DDS delivers polished component (1-2 weeks) - swap is trivial since interface matches' },
    ].map((item, idx) => (
      <div key={idx} style={{
        display: 'flex',
        alignItems: 'flex-start',
        gap: '12px',
        padding: '12px',
        background: SLATE[50],
        borderRadius: RADIUS.md,
      }}>
        <div style={{
          width: '28px',
          height: '28px',
          borderRadius: RADIUS.full,
          background: DEEP_CURRENT[500],
          display: 'flex',
          alignItems: 'center',
          justifyContent: 'center',
          color: PRIMITIVES.white,
          fontSize: '13px',
          fontWeight: 700,
          flexShrink: 0,
        }}>{item.step}</div>
        <div>
          <div style={{ fontSize: '13px', fontWeight: 600, color: ABYSS[500] }}>{item.action}</div>
          <div style={{ fontSize: '12px', color: DUSK_REEF[500] }}>{item.desc}</div>
        </div>
      </div>
    ))}
  </div>
</div>

<InfoBox variant="warning">
  <strong>Why does this work?</strong> Because you're building to DDS's interface from day one,
  the final swap is a 1-line import change, not a rewrite. The interface contract ensures
  compatibility. DDS delivers the polished, tested, accessible version within 1-2 weeks.
</InfoBox>
