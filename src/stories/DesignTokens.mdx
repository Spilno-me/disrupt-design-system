## üéØ Best Practices

1. **Use Tailwind classes for static styling** - Faster, cached, optimized
2. **Use tokens for dynamic values** - When values depend on state/props
3. **Follow the design system** - Don't create custom values
4. **Be consistent** - Use the same tokens across products
5. **Reference documentation** - Check this page when unsure
